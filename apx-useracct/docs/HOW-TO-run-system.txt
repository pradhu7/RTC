
To run the dropwizard services:

bash> java -jar dw-tokenizer/target/apixio-rest-tokenizer-0.0.1-SNAPSHOT.jar server dw-tokenizer/src/main/resources/tokenizer.yaml
bash> java -jar dw-useracct/target/apixio-rest-useracct-0.0.1-SNAPSHOT.jar   server dw-useracct/src/main/resources/user-account.yaml

(replace 0.0.1-SNAPSHOT as appropriate).
